\begin{abstract}
\addcontentsline{toc}{section}{Resumo}

Neste trabalho foi realizada a análise prática de circuitos moduladores e demoduladores digitais BPSK. Para o modulador, foi utilizado o módulo MCA 8801 - DataPoll, configurado para operar com sinal de entrada em formato de onda quadrada e empregando o modulador FM, sendo este utilizado como modulador digital. O demodulador foi construído com um circuito LM 565 que opera como um PLL para recuperar a informação recebida. Durante o experimento, foi possível observar o funcionamento de um esquema de modulação FSK binário.
\end{abstract}
